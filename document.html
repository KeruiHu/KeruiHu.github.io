Request for Proposal.

At the Best SENG Painters Company, we are looking for a software application that can store customer information including past orders. Additionally, customers should be able to order remotely, but due to our limited company size we will not be processing payments through the app. When the order is placed, feedback should be given about which store the order was placed at, and an estimated time of order completion. Some other features we are looking for are an AI colour matching algorithm and a colour visualizer, but these are not the main focuses of the application and should come secondary when developing the final product.
View full request for proposal

Constraint #1

The application should not interfere with employee workflow and the application must interface with the current Point of Sale system

Constraint #2

The application cannot purchase directly through the current Point of Sale system and all sales must be done in-store

Constraint #3

The application must follow a linear path of questions for selecting products
